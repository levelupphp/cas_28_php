# cas_28_php

# Installation

## Command prompt:

### enter htdocs

cd c:\xampp\htdocs

### clone project

git clone [project-url]

### enter project folder

cd cas_28_php

### install vendor libraries

composer install

### create env file

copy .env.example to .env file (and change values for database)

### generate application key

php artisan key:generate

### start server

php artisan serve
